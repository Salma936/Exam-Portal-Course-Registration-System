# 📚 Exam Portal & Course Registration System

A responsive web-based **Exam Portal & Course Registration System** that enables students to securely register for courses, manage exam-related activities, and interact with a user-friendly interface. The application is built using **HTML, CSS, and JavaScript** with backend API integration for authentication and data management.

---

## 🚀 Features

* 🔐 Secure user authentication
* 📝 Course registration and management
* ✅ Client-side form validation
* 🔄 Dynamic content rendering
* 📱 Responsive UI for desktop and mobile devices
* 🌐 Backend API integration
* ⚡ Optimized user experience and performance

---

## 🛠️ Tech Stack

**Frontend**

* HTML5
* CSS3
* JavaScript (ES6)

**Backend**

* REST APIs

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/exam-portal.git
```

2. Navigate to the project folder

```bash
cd exam-portal
```

3. Open `index.html` in your browser or use a local server.
